API Endpoints

Autenticacao

Metodo Endpoint Descricao
POST /api/auth/register Registrar novo usuario
POST /api/auth/login Login (retorna JWT)
POST /api/auth/refresh Renovar access token
GET /api/auth/me Usuario atual

Busca (Jackett)

Metodo Endpoint Descricao
GET /api/search?query=&type=&categories= Buscar torrents
GET /api/search/indexers Listar indexadores

Downloads (qBittorrent)

Metodo Endpoint Descricao
GET /api/downloads Listar downloads
POST /api/downloads Iniciar download
GET /api/downloads/:hash Detalhes do download
POST /api/downloads/:hash/pause Pausar download
POST /api/downloads/:hash/resume Retomar download
DELETE /api/downloads/:hash Remover download
GET /api/downloads/:hash/files Arquivos do torrent

Streaming

Metodo Endpoint Descricao
GET /api/stream/:hash/:fileIndex Stream de arquivo (suporta range requests)

Series (Sonarr) & Filmes (Radarr)

Metodo Endpoint Descricao
GET /api/series Listar series
POST /api/series Adicionar serie
GET /api/series/lookup?term= Buscar serie
GET /api/movies Listar filmes
POST /api/movies Adicionar filme
GET /api/movies/lookup?term= Buscar filme

Dashboard

Metodo Endpoint Descricao
GET /api/dashboard Dados consolidados (downloads, series, filmes, disco)
